🛡️ الحارس الدلالي (Semantic Guard)
نظام الذكاء الاصطناعي للتدقيق المنطقي والارتقاء بجودة البيانات اللحظية
🌟 نظرة عامة (Overview)
الحارس الدلالي هو نموذج أولي (POC) لمساعد ذكي  لجودة البيانات. يتميز بقدرته على الانتقال من "التحقق الرقمي" إلى "الإدراك المنطقي"، حيث يستخدم الذكاء الاصطناعي لرصد التناقضات السياقية (مثل تعارض المهنة مع السن) لحظياً أثناء جمع البيانات الميدانية.
🚀 التشغيل السريع (Quick Start)
1. التشغيل السحابي (Cloud Access) - الخيار الأسرع
يمكنكم تجربة النظام مباشرة دون الحاجة لأي تثبيت عبر الرابط التالي: https://semantic-guard-poc.streamlit.app/ 👉 رابط النسخة الحية على Streamlit
ملاحظة: يدعم الرابط وضع المحاكاة (Demo Mode) تلقائياً، ويمكن تفعيل الذكاء الاصطناعي الحقيقي عبر الشريط الجانبي.
2. التشغيل المحلي (Local Execution)
تثبيت المكتبات:
pip install -r requirements.txt


إعداد المفتاح: ضع مفتاح OpenAI في ملف .env (اختياري).
التشغيل:
python -m streamlit run src/demo.py
