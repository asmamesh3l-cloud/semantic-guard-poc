# semantic-guard-poc
🛡️ الحارس الدلالي (Semantic Guard)

نظام الذكاء الاصطناعي للتدقيق المنطقي والارتقاء بجودة البيانات اللحظية

🌟 نظرة عامة (Overview)

الحارس الدلالي هو نموذج أولي (POC) لمساعد ذكي يهدف إلى ثورة جودة البيانات الميدانية. بدلاً من الاعتماد على قواعد التحقق التقليدية الجامدة، يستخدم النظام النماذج اللغوية الكبيرة (LLMs) لتحليل العلاقات السياقية والمنطقية بين الإجابات لحظياً، مما يضمن اكتشاف الأخطاء المعقدة (مثل تعارض المسمى الوظيفي مع العمر) في "لحظة الحقيقة" أثناء جمع البيانات.

🚀 المميزات الرئيسية (Key Features)

التدقيق السياقي: اكتشاف التناقضات الدلالية التي تعجز عنها البرمجة التقليدية.

درجة الثقة (Confidence Score): منح كل سجل تقييماً احتمالياً لمدى معقوليته.

تنبيهات لحظية: تقديم تفسيرات بشرية فورية للباحث الميداني لتصحيح الخطأ فوراً.

بنية مرنة: سهولة التكامل مع منصات الاستبيانات عبر واجهة برمجة التطبيقات (API).

👥 فريق العمل: الابتكار التصميمي

نحن فريق متخصص يسعى لرفع جودة البيانات الوطنية من خلال الابتكار التقني:

اسماء العقيدي

ساره الغامدي

سعيده الأكلبي

صباح القحطاني

صفيه العثمان

🛠️ التقنيات المستخدمة (Tech Stack)

اللغة الأساسية: Python

الذكاء الاصطناعي: GPT-4o / Llama-3 (عبر LangChain)

الواجهة الخلفية (Backend): FastAPI

واجهة التجربة (Demo): Streamlit

تحليل البيانات: Pandas & KNIME

⚙️ التنصيب والتشغيل (Setup & Installation)

1. المتطلبات

تأكد من وجود Python 3.9+ ومفتاح API صالح (OpenAI/Groq).

2. التثبيت

git clone [https://github.com/asmamesh3l-clod/semantic-guard-poc.git](https://github.com/asmamesh3l-clod/semantic-guard-poc.git)
cd semantic-guard-poc
pip install -r requirements.txt



3. الإعداد

قم بإنشاء ملف .env وأضف مفتاح الـ API الخاص بك:

API_KEY=your_key_here
LLM_MODEL=gpt-4o



4. التشغيل

لتشغيل المحرك (Backend):

uvicorn src.main:app --reload



لتشغيل واجهة العرض (Demo):

streamlit run src/demo.py
